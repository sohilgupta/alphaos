# Theme: India Data Center Stack

## Core Thesis
India has a listed-market value chain for the AI data-center buildout. The local stack spans compute, design/build, connectivity, power, backup, cooling, cabling and cybersecurity.

This creates an India-specific expression of the [[AI-Infrastructure-Capex-Supercycle]]: if global inference demand keeps compounding, domestic sovereign AI, hyperscaler leases and IndiaAI Mission procurement can pull demand through multiple listed suppliers.
